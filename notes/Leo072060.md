# Uniswap

## 本质

Uniswap 是 [去中心化交易所](Decentralized Exchange)（DEX）的代表，由 Hayden Adams 于 2018 年创立。

核心创新是 **AMM（自动做市商）**：不靠订单簿撮合，而是用一个数学公式，由资金池的储备比例**连续决定**价格。

## 恒定乘积公式

```text
x * y = k
```

-   `x`、`y` = 池中两种代币的数量（如 ETH、USDC）
    
-   `k` = 常数，每笔交易后必须保持不变（忽略手续费）
    

价格 = 池子储备比例。单笔交易相对池子越大，价格偏移越明显（**滑点 / 价格冲击**）。若池内价格偏离外部市场，套利交易会把它拉回。

## 例子：用 1 ETH 换 USDC

```text
交易前：10 ETH,  5000 USDC     → k = 10 × 5000 = 50000
存入 1 ETH → 池变 11 ETH
为保持 k：USDC = 50000 / 11 = 4545.45
换得 USDC = 5000 − 4545.45 = 454.55
兑换价 = 454.55 USDC / ETH
```

现货价本是 500 USDC/ETH，实际只换到 454.55——差的部分就是**价格冲击**。

## 流动性池与 LP

把资金存入池子就成为**流动性提供者（LP）**，别人交易时用你的资金兑换，你按份额分手续费。

-   手续费以 **0.3%** 为例，**按输入代币计价、直接留在池里**分给 LP。
    
-   例：别人用 1 ETH 换 USDC → 手续费 0.003 ETH（≈1.5 USDC）；你占池 10% → 分到 0.0003 ETH。
    

LP 真实收益 ≠ 手续费

还要减去**无常损失**，并受交易量、价格波动、Gas 成本影响。

## 提供流动性：进入与退出

**进入（添加流动性）**

```text
1. 按池子当前比例，同时存入两种代币（V2 为 50/50 等值）
   例：池价 500 USDC/ETH → 按 1 ETH : 500 USDC 存
2. 收到 LP token（份额凭证，V2 是 ERC-20「UNI-V2」），代表你占池子的比例
3. 别人交易的 0.3% 手续费留在池里 → 你份额对应的资产随之变多
```

**退出（撤出流动性）**

```text
1. 销毁（burn）LP token
2. 按「退出时」的池子比例取回两种代币 + 累计手续费
```

取回按退出时比例，不是存入时

正因如此才有**无常损失**——若期间两币相对价格变了，取回的两币数量会和存入时不同。

## 无常损失（Impermanent Loss）

AMM 会自动再平衡，所以当两币**相对价格变动**时，你 LP 持仓的价值会**低于**当初直接拿着两币（HODL）的价值。

-   价格若**回到存入时的比例**，这个损失消失 → 所以叫"无常 / 暂时"。
    
-   手续费收入可能补上这个差、也可能补不上。价格波动越大，无常损失越大。
    

## 无常损失·数学推导

设**存入时**价格 `p₀`（X 以 Y 计价），此刻你投入的两种代币数量记为 `x₀`、`y₀`（下标 0 = 存入时），`k = x·y`。池子任意时刻满足 `x·y = k` 且边际价 `y/x = p`，联立解出：

```text
x = √(k/p)        y = √(k·p)
```

-   代入存入价 `p₀` 即得你的初始投入：`x₀ = √(k/p₀)`、`y₀ = √(k·p₀)`。
    

金额任意，但比例不能任意

你能自由选**投多少**（规模），但两币必须按**当前价比例**投（V2 两边等值）；比例不对合约会退回多余的一边。  
上面的 `x₀=√(k/p₀)` 是"**你 = 整个池**"简化下的**池子储备**。真实里你投 `(Δx, Δy)` 只需满足 `Δy/Δx = p₀`，规模任意，份额 `s = Δx / 池子总X`。  
因为**无常损失是百分比、与投入规模无关**（规模一约分就没了），用这个简化不影响结论。

-   而不带下标的 `x,y` 是价格 `p` 时的**当前**池中数量——**你退出时按份额拿回的正是它们**。价格一变，AMM 就再平衡，所以 `x ≠ x₀`。
    

分别算两种价值（都以 Y 计价）；`V_hold` 表示"若把这 `x₀,y₀` 原封不动持有到价格 `p` 时的价值"：

```text
LP 仓位：  V_pool = y + p·x = 2√(k·p)
直接持有： V_hold = y₀ + p·x₀ = √k · (p₀ + p) / √p₀
```

**关键**：`V_pool` 用**当前** `x,y`（币数被 AMM 改过），`V_hold` 用**初始** `x₀,y₀`（HODL 从没动）；两边都按当前价 `p` 估值。差别正来自"币数变没变"。

令价格变动倍数 `r = p/p₀`，两者相除得到无常损失：

```text
V_pool / V_hold = 2√r / (1 + r)

IL = 2√r/(1+r) − 1     （≤ 0，仅 r=1 时为 0）
```

**为什么恒 ≤ 0**：由均值不等式 `1 + r ≥ 2√r`（r=1 取等），故 `2√r/(1+r) ≤ 1`。

**几何本质**：`V_pool = 2√(k·p)` 是关于 p 的**凹函数**，`V_hold` 是 p 的**直线**且恰在 `p=p₀` 处与它相切 → 凹曲线永远在切线下方 → LP 价值恒 ≤ HODL。

**直觉**：AMM 逼你"卖涨买跌"——X 涨时替你卖 X、X 跌时替你买 X，你总在错的一边；套利者把池价拉回市价时提取的价值，正是无常损失。

| 价格变动 r | 无常损失 |
| --- | --- |
| 1×（不变） | 0 |
| 1.25× | −0.6% |
| 2× | −5.7% |
| 4× | −20% |
| 5× | −25.5% |
| 0.5×（腰斩） | −5.7% |

涨跌对称：`r` 与 `1/r` 的无常损失相同（如 2× 与 0.5× 都是 −5.7%）。

## 退出时到底拿回多少（份额 s）

上面为简化把"你 = 整个池"。真实中你只占一部分，用**份额 s** 折算：

```text
份额 s = 你的 LP token / 总 LP token   （存入时定下，别人交易不改变 s）
拿回 X = s · x_now      拿回 Y = s · y_now
其中 x_now = √(k/p),  y_now = √(k·p)   （当前价 p 下的池子储备，已含累积手续费）
```

**完整算例**（正好复现上表 r=4 → −20%）：

```text
存入：池子 10 ETH / 5000 USDC（p₀=500），你投入 1 ETH + 500 USDC
  → 新池 11 ETH / 5500 USDC，k = 11×5500 = 60500，份额 s = 1/11

价格涨到 2000（4×），套利再平衡（k 不变）：
  x_now = √(60500/2000) = 5.5 ETH
  y_now = √(60500×2000) = 11000 USDC      （5.5 × 11000 = 60500 ✓）

按 s = 1/11 退出：
  拿回 ETH  = 5.5 / 11   = 0.5 ETH
  拿回 USDC = 11000 / 11 = 1000 USDC
  LP 价值   = 0.5×2000 + 1000 = 2000 USDC

对比 HODL：原 1 ETH + 500 USDC → 1×2000 + 500 = 2500 USDC
  无常损失 = 2000 / 2500 − 1 = −20%
```

看点：你**存进 1 ETH，只取回 0.5 ETH**（外加更多 USDC）——价格涨时 AMM 替你**卖掉了一半 ETH**，这就是"卖涨"的实体，也是那 −20% 的来源。
